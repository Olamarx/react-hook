#Search Image Project

In this project, I: 
- Created an Accordion for FAQ.
- Created an app for a search which uses Wikipedia API
- Created a dropdown which listens to outside clicks
- Created a translator app which can translate any language to the options provided. It uses google API.

## Getting Started

To run a local copy of this project follow these simple steps.

### Prerequisites

- Any web browser (e.g. Google Chrome)
- Any Code Editor (e.g. Vscode)

### Cloning the repository

- Create a new folder.
- Open your console in the new folder and run this command: `https://github.com/Olamarx/react-hook.git`.
- Run this command on your terminal to install all the dependencies: `npm i`.
- Finally, run this command to start a local server of the project: `npm start`.
